# SleepyMusics
Play sleepy musics, good night~ :sleeping: :zzz:
<hr>
<p><strong>Why:</strong></p>
因为我的iphone是乞丐版16g，已经被各种必备的软件塞满了，音乐播放器实在装不了，更别提这么多音乐文件了。:cry: <br/>
但是睡前已经养成习惯要伴着睡眠音乐入睡，所以写个播放器放在旧安卓手机上播放。:grinning: <br/>
恩？为什么不直接安装个虾米？因为旧安卓手机实在卡成狗，点什么都特别卡，还是写个cute tiny的apk吧～ :stuck_out_tongue: <br/>

<p><strong>What:</strong></p>
只有一个界面：<br/>
<img src='https://raw.githubusercontent.com/FrannyZhao/SleepyMusics/master/source/app_main_screen.png' width="300px" style='border: #f1f1f1 solid 1px'/>
<br/>
点击下方的按钮即可开始播放12首入眠音乐。大约2小时10分钟，声音调到若即若离的大小，一般半小时就睡着了。<br/>
播放完这12首音乐会自动停止。<br/>
